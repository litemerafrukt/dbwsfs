Denna sajt är skapad som slutprojekt i kursen [Webbaserade ramverk 1 7,5p](https://dbwebb.se/kurser/ramverk1) på Blekinge Tekniska Högskola.

## Installation
1. Klona repot.

Det är allt.

För utveckling kör
1. `make install`
2. `npm install`

Sajten använder en sqlite-databas, se `db/setup-db.sql` för schema.
